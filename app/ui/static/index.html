Архитектура MVP

UI / API

FastAPI принимает запросы, показывает документацию и даёт единый HTTP-контур.

Inventory

Mock inventory и NetBox adapter формируют источник устройств и атрибутов.

Automation Core

Nornir orchestration, профили, selector, compliance и batch-операции.

Execution

Mock backend для стенда и Netmiko backend для реального виртуального роутера.

State / Audit

SQLite, JSON state, snapshots, operation journal, jobs и Celery workflow.
